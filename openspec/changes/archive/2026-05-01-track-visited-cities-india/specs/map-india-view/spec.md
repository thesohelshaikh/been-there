## ADDED Requirements

### Requirement: Interactive Map of India
The system SHALL display an interactive map focused on India.

#### Scenario: Display Map on Launch
- **WHEN** the user opens the application
- **THEN** the system SHALL display a map centered on the geographic bounds of India

### Requirement: State Boundary Rendering
The map SHALL display distinct boundaries for all Indian states and union territories.

#### Scenario: View State Outlines
- **WHEN** the map is rendered
- **THEN** the system SHALL draw visible borders separating different states of India
